# DeeJay 

DeeJay is a full-stacked application that uses the RRRAM stack (and yes, I made it up myself), consists of Rust, Redis, React, Actix, and MongoDB. The application allows users to login to their spotify accounts and host a live AUX session, which allows other users to join and request songs to the queue. Another use case is the live LISTEN-TOGETHER session, which allows multiple users to sync up their music players together.

# Installation

### With Docker

You can install and run the application using Docker, this is the recommended method. Run the following command in the root directory.

```bash
sudo docker compose up
```
